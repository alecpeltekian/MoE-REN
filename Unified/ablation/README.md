# Ablation Analysis

## Usage

Run single fold ablation:

```bash
cd ablation
bash run_ablation_example.sh
```

Run full ablation (all folds):

```bash
bash run_full_ablation.sh
```

## Scripts

- `run_ablation_example.sh` - Run ablation for single fold
- `run_full_ablation.sh` - Run ablation for all folds
- `ablation_analysis.py` - Main ablation analysis script
- `visualize_ablation.py` - Generate visualizations
- `summarize_ablation.py` - Generate summaries
